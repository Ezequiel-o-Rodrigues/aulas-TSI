Elementos visuais na web

O que são Imagens?

Imagens são elementos visuais que enriquecem o conteúdo do site. São inseridas com a tag <img> e são essenciais para comunicação visual.

Formatos de Imagem

  • JPG/JPEG: Fotos e imagens complexas (com compressão)
  • PNG: Imagens com transparência, logos
  • GIF: Animações simples
  • SVG: Gráficos vetoriais (escaláveis sem perder qualidade)
  • WebP: Formato moderno com melhor compressão

Atributos Importantes

  • src: Caminho da imagem (obrigatório)
  • alt: Texto alternativo para acessibilidade
  • width/height: Dimensões da imagem
  • loading="lazy": Carregamento sob demanda

♿ Acessibilidade

Sempre use o atributo alt para descrever a imagem. Isso ajuda:

  • Pessoas com deficiência visual (leitores de tela)
  • Quando a imagem não carrega
  • SEO (mecanismos de busca)

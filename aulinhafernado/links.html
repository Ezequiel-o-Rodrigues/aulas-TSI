O que são Links?

Links (âncoras) são elementos que permitem navegação entre páginas, seções ou recursos externos. São a base da web.

Tipos de Links

  • Links Externos: Levam para outros sites (href="https://...")
  • Links Internos: Navegam entre páginas do mesmo site
  • Âncoras: Pulam para seções da mesma página (href="#secao")
  • Links de Email: Abrem cliente de email (href="mailto:...")
  • Links de Telefone: Iniciam chamada (href="tel:...")

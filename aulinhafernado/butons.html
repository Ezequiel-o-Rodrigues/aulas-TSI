Elementos interativos essenciais

O que são Botões?

Botões são elementos clicáveis que executam ações quando acionados. São fundamentais para interação do usuário com o site.

Tipos de Botões

  • button - Botão genérico para ações JavaScript
  • submit - Envia dados de formulário
  • reset - Limpa campos do formulário
